# 🤰 Pregnancy Risk Checker

A simple machine learning web app that predicts pregnancy risk based on user inputs.

## 🚀 Features
- User-friendly interface (no medical knowledge needed)
- Machine learning model (XGBoost)
- Instant risk prediction (Low / High)
- Simple health suggestions

## 🧠 Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- XGBoost

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
